# Bartr - Modern Peer-to-Peer Trading Platform

A vibrant, engaging platform for trading items, skills, and experiences without cash.

## Features

- **Browse & Trade**: Discover items and services from your community
- **Rideshare Forum**: Find rides and share costs with local traders
- **Profile Customization**: Showcase your skills and items
- **Friend System**: Connect with other traders
- **AI Matching**: Smart recommendations based on your interests
- **Notifications**: Stay updated on trades and messages

## Development

This is a static website built with:
- HTML5
- Tailwind CSS (via CDN)
- Vanilla JavaScript

To run locally:
```bash
npx serve
```

Then open http://localhost:3000
